# Grout-Platform - Current Features Overview

## Project Summary
`grout-platform` is a Python-based project containing multiple applications and tooling for data, ML, and "road-ready" experiments. Legacy "Pungi" naming has been replaced by `grout-platform` branding.

## Project Structure
```
grout-platform/
├── apps/                    # Applications directory
│   ├── roadready-api/       # Backend/API for road-ready features
│   └── roadready-ui/        # Frontend/UI (Expo/React Native)
├── data/                    # Datasets and data assets
├── notebooks/               # Exploration and experiments
├── src/                     # Core Python source code
│   ├── covid19/            # COVID-19 related tools
│   └── mypro/              # Personal projects/tools
├── assets/                  # Images and static assets
├── road-ready/              # Virtual environment for road-ready stack
├── vs-code-extention/       # VS Code extension
├── main.py                  # Main entry point
└── README.md                # Project documentation
```
