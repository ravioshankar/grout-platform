## 🚀 How to Run

### Quick Start
```bash
cd apps/roadready-ui
npm start
```

Then press:
- `w` for Web
- `a` for Android
- `i` for iOS

### Platform-Specific
```bash
npm run web      # Web
npm run android  # Android
npm run ios      # iOS (macOS only)
```

### Verification
```bash
node verify-app.js
```
