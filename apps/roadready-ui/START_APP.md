# 🚀 Quick Start - RoadReady UI

## One-Command Start

```bash
cd apps/roadready-ui && npm start
```

Then press:
- **`w`** for Web
- **`a`** for Android
- **`i`** for iOS

---

## Platform-Specific Commands

### 🌐 Web (Fastest)
```bash
npm run web
```
Opens at: http://localhost:8081

### 🤖 Android
```bash
npm run android
```
Requires: Android Studio + Emulator

### 🍎 iOS (macOS only)
```bash
npm run ios
```
Requires: Xcode + Simulator

---

## 📱 Test on Physical Device

### Using Expo Go (Easiest!)

1. **Install Expo Go app:**
   - [iOS App Store](https://apps.apple.com/app/expo-go/id982107779)
   - [Android Play Store](https://play.google.com/store/apps/details?id=host.exp.exponent)

2. **Start dev server:**
   ```bash
   npm start
   ```

3. **Scan QR code** with your phone

4. **App loads instantly!** ✨

---

## 🔧 First Time Setup

```bash
# 1. Navigate to project
cd apps/roadready-ui

# 2. Install dependencies
npm install

# 3. Start development server
npm start

# 4. Choose platform (press key in terminal)
# w = Web, a = Android, i = iOS
```

---

## 🐛 Troubleshooting

### Clear Cache
```bash
npx expo start -c
```

### Reinstall Dependencies
```bash
rm -rf node_modules
npm install
```

### Reset Project
```bash
npm run reset-project
```
