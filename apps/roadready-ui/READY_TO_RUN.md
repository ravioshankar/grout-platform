## 🎨 Customization

### Change App Name
Edit `app.json`:
```json
{
  "expo": {
    "name": "Your App Name"
  }
}
```

### Change Theme Colors
Edit `constants/theme.ts`:
```typescript
export const Colors = {
  light: {
    primary: '#your-color'
  }
}
```
